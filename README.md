## Requirements:

To run the application you will need the following:
* [Node](https://nodejs.org/)
* [Yarn](https://yarnpkg.com/) or [npm](https://www.npmjs.com/)
* [React](https://reactjs.org/)

### How to run:
#### Install the dependencies
```sh
yarn
```
#### build and start the application
```sh
yarn start
```
